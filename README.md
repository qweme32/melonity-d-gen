## Melonity Docs Generation

1.  Install dependencies
    ```sh
    npm install
    ```
1.  Build via swc
    ```sh
    npm run build
    ```
2.  Transpile declaration into json
    ```sh
    npm run start <file>
    ```

    #### For melonity.d.ts
    ```sh
    npm run start melonity
    ```
3. Explore \<file>.d.json
